# Gemini API Key Setup

Set your Gemini API key in your environment before running the dashboard AI assistant.

## Mac/Linux

```bash
export GEMINI_API_KEY="your_api_key_here"
```

## Windows (Command Prompt)

```bat
setx GEMINI_API_KEY "your_api_key_here"
```

After setting the variable, restart your terminal (and VS Code terminal) so the new environment variable is available.
